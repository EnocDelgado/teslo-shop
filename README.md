# Teslo API


## Steps

1. Clone project.

2. install modules:
```
yarn install
```

3. Clone ```.env.template``` file and rename as ```.env```.

4. Chnage environments variables.

5. Up database
```
docker-compose up -d
```

6. Run seed:
```
http://localhost:3000/api/seed
```

7. Run dev mode:
```
yarn start:dev
```
